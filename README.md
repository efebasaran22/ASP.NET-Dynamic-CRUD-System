# 🌐 ASP.NET Dinamik CRUD ve Veritabanı Yönetim Sistemi

Bu proje, ASP.NET Web Forms mimarisi ve MS SQL Server kullanılarak geliştirilmiş, veri tabanı işlemlerini (CRUD) web arayüzü üzerinden yönetmeyi sağlayan dinamik bir web uygulamasıdır.

### 🚀 Projenin Özellikleri
* **Kimlik Doğrulama:** Güvenli kullanıcı kayıt (Register) ve giriş (Login) sistemi.
* **Dinamik Tablo Yönetimi:** Kullanıcıların arayüz üzerinden veritabanında anlık olarak yeni tablolar oluşturabilmesi.
* **CRUD Operasyonları:** Oluşturulan tablolara veri ekleme, listeleme, güncelleme ve silme işlemleri.
* **Modüler Tasarım:** MasterPage yapısı kullanılarak tutarlı ve yönetilebilir bir web arayüzü sunulması.

### 🛠️ Kullanılan Teknolojiler
* **Backend:** C#, ASP.NET Web Forms (.NET Framework 4.7.2)
* **Veritabanı:** Microsoft SQL Server (ADO.NET / SqlClient)
* **Frontend:** HTML, CSS, Web Forms UI Kontrolleri

### 💡 Geliştirici Notu
Bu çalışma, web tabanlı uygulamalarda veritabanı mimarisini anlamak, güvenli veri akışı sağlamak ve dinamik kullanıcı arayüzleri kurgulamak amacıyla geliştirilmiş kapsamlı bir staj/ödev projesidir.
